# User Management API

This is a simple Node.js API for managing users. It allows for inserting and fetching user data.

## Features
- Fetch all users
- Add a new user

## Requirements
- [Node.js](https://nodejs.org/) (v14 or higher)
- [npm](https://www.npmjs.com/) (Node Package Manager)
